Aprendendo a organizar o seu CSS

Diga adeus aos dias de "tentativa e erro", iremos desvendar todas as entrelinhas do CSS de forma direta e objetiva, garantindo que o conhecimento não irá se perder. Nosso objetivo não é formar desenvolvedores para "apenas" disputar vagas no mercado de trabalho, mas sim, ajuda-los a encontrar seu caminho para as estrelas.

Objetivo

O curso tem como objetivo desmistificar todos os assuntos relacionados a organização do CSS, desde técnicas para definir a melhor nomenclatura para para suas classes até criar componentes totalmente funcionais e desacoplados. Abordamos os principais conceitos relacionados a escalabilidade e mantenabilidade, passando por todas as principais metodologias mas não necessariamente iremos nos ater a elas. O objetivo é gerar através de conhecimento e muita prática um senso crítico que possibilite tomar a melhor deciscão para cada caso.

Como funciona?

É 100% online, não temos a quatidade exata de aulas e horas definidas, mas iremos cobrir toda a ementa com calma porém de forma direta, para isso temos cada vídeo com uma duração média de 5 à 15 minutos, para tornar o aprendizado o mais dinâmico possível. No final do curso você irá receber um certificado comprovando todas as horas de estudo, apesar de nós dois sabermos que o importante será o conhecimento adquirido. ;)
